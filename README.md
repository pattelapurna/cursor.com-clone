# cursor.com-clone

Static clone of the Cursor.com. is a UI clone 
[![Netlify Status](https://api.netlify.com/api/v1/badges/3d8c438d-fa14-45ed-bb36-0639c33bd968/deploy-status)](https://app.netlify.com/projects/cursor-ui-ppc/deploys)

## What’s Here
- `index.html` — page markup
- `styles.css` — site styles
- `asserts/` — images and other static files
- `fonts/` — local font files

## Run Locally
You can open the file directly:
- Double click `index.html`

Or serve it locally (recommended for relative paths and fonts):

```bash
git clone https://github.com/pattelapurna/cursor.com-clone.git
```

Then visit `/path/to/cursor.com-clone/index.html`.

## Project Notes
- No build step required.
- No dependencies.
