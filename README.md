# cursor.com-clone

Static clone of the Cursor marketing site.

## What’s Here
- `index.html` — page markup
- `styles.css` — site styles
- `asserts/` — images and other static files
- `fonts/` — local font files

## Run Locally
You can open the file directly:
- Double click `index.html`

Or serve it locally (recommended for relative paths and fonts):

```bash
git clone https://github.com/pattelapurna/cursor.com-clone.git
```

Then visit `/path/to/cursor.com-clone/index.html`.

## Project Notes
- No build step required.
- No dependencies.
